#  Geo Bash

A rework of Geometry Dash

**How it works:** GamePlay is Simple as the Geometry Dash Game, here you will just crash out alot that is one thing.
 
---
 
## Controls
- Space: Jump
  
---
 
## Features
- Bg Color Change
- World Movement
- Death
- Traps
- Death Effects
- SFX
